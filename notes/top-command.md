# Top Command Notes

## Purpose
The `top` command is used for real-time system monitoring.

## What It Shows
- CPU usage
- Memory usage
- Running processes
- Load average
- Process IDs (PID)

## Useful Keys

| Key | Function |
|---|---|
| P | Sort by CPU |
| M | Sort by Memory |
| k | Kill process |
| r | Renice process |
| q | Quit top |

## Example

```bash
top
```

## Learning
Used for identifying high CPU or memory consuming processes.
